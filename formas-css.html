Formas de incluir CSS em uma página HTML

1. CSS inline: é escrito direto dentro da tag, usando o atributo style.

2. CSS interno: é escrito dentro da tag style, que fica no head do documento.

3. CSS externo: fica em um arquivo separado, normalmente com extensão .css, ligado ao HTML pela tag link.

Exemplo de CSS externo: <link rel="stylesheet" href="estilo.css">
